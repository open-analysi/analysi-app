# Agents

```
agents/
  source/   Edit here. Golden copies of prod agents managed in this repo.
  dev/      Dev-only agents (never shipped). For local Claude Code use.
  dist/     Build output. Generated by `make package-agents` from skilltree.yml.
            Do not edit — contents are overwritten on each package run.
            COPY'd into Docker image at build time.
```

**Adding a new prod agent:** add it to `skilltree.yml` under `dependencies:`, run `skilltree install && make package-agents`, commit `agents/source/` and `agents/dist/`.

**Adding a dev-only agent:** add it to `skilltree.yml` under `dev-dependencies:`, place the file in `agents/dev/`.

See `scripts/agents_management/CLAUDE.md` for full packaging docs.
